===== Functionality

and removing your finger from the sensor will turn the LED light off.

===== YAML Pin Order
The order for declaring the pin for the LED is as follows:

[source, yaml]
----
digital-output:
  led2:
    address: 26
----
...

The order for declaring the pin for the push button is as follows:

[source, yaml]
----
digital-input:
  touch-switch-input:
    address: 17
----
...
